##Description of files 

##colo1_plm.rds: log fold changes of HT29 vs plasmid, using normalised counts
##colo1_plm_scale.rds: log fold changes of HT29 vs plasmid, using normalised counts and scaled using negative and positive controls (0 and -1 respectively)
##colo1_c91.rds: log fold changes of HT29 vs parental(cas9 negative), using normalised counts
##IDs_colo1.rds: row annotation for all previous tables
##colo1_plm_GEN.rds: log fold changes of HT29 vs plasmid, using normalised counts, aggregated at gene level
##COLO1_RUNMERGED_EXACT_ANNOTATED.txt: raw counts for HT29, parental lines (cas9 negative) and plasmid
##lib-COLO-1.stats: stats metrics for the plasmid
##All.stats:  stats metrics for the plasmid, parental lines (cas9 negative) and HT29
##paralog_scores_ryan.csv: tables with scores to determine parlous from Ryan et al (https://doi.org/10.1016/j.cels.2021.08.006)
##SCORE_median_lfc.rds: log fold changes aggregated at gene level using the median of all guides for HT29 downloaded from project score (https://score.depmap.sanger.ac.uk/)
